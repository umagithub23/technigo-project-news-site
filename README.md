# News Site

- This assignment was about building a simple news‑style webpage using only HTML and CSS. The goal was to practice structuring content with semantic HTML and creating a visually appealing layout using modern CSS techniques. The project includes a header, navigation, content sections, images, and a footer, styled to resemble a clean, responsive news site.

- I started by planning the overall structure of the page using semantic HTML elements such as header, nav, main, section, article, and footer. Once the structure was in place, I focused on layout and styling using:
  - CSS Grid for arranging the main content and news cards
  - Flexbox for aligning items inside the header, navigation bar, and footer
  - Hover effects to make the page feel interactive
  - Simple animations to add smooth transitions
  - Clickable links that navigate to external news sites
  - Consistent spacing, typography, and image styling to create a clean visual hierarchy
I worked iteratively — building one section at a time, testing responsiveness, and refining the layout as I went

- If I had more time to continue improving this project, I would focus on:
  - Expanding the site with additional news categories or sub‑pages
  - Refining typography and spacing for a more polished, editorial look
  - Adding links for home, contact, about
